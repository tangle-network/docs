# Integrating with Restaking

To integrate with Tangle restaking (for building a dApp, indexer, or integration), you interact with Tangle’s **EVM protocol contracts**.

Key contracts:

- `Tangle`: blueprint registration, service requests, jobs, payments, and slashing coordination.
- `MultiAssetDelegation`: deposits, delegation, operator registration/lifecycle, and withdrawal scheduling.
- `LiquidDelegationFactory` / `LiquidDelegationVault`: liquid restaking via transferable vault shares over delegated positions.

## ABIs / Integration Approach

Integrations typically use one of:

- Ethers/viem + published ABIs
- Foundry `cast` for scripting
- An indexer that listens to protocol events for operator/service activity

Once launch endpoints are published, update your integration to point to the correct RPC and deployed contract addresses.

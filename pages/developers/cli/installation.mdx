# Tangle CLI Installation

## Prerequisites

1. Install Rust

You can install Rust by following the instructions on the [Rust Installation Guide](https://www.rust-lang.org/tools/install)

or run the following command:

```shell
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Install `cargo-tangle`

### From source (recommended for now)

```shell
cargo install cargo-tangle --git https://github.com/tangle-network/blueprint --branch v2 --force
```

### Verify

```shell
cargo tangle --help
```

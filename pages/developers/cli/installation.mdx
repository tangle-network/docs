# Tangle CLI Installation

## Prerequisites

1. Install Rust

You can install Rust by following the instructions on the [Rust Installation Guide](https://www.rust-lang.org/tools/install)

or run the following command:

```shell
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Install from source

1. From source (nightly)

> The script supports Linux, MacOS, and Windows (WSL2)

```shell
cargo install cargo-tangle --git https://github.com/tangle-network/blueprint --force
```

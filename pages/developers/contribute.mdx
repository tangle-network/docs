# Contribute

All of Webb's technology is open source and we invite researchers, developers and enthusiasts to participate in creating a world of more natural privacy.

Our mission **is** privacy and security. We encourage all users to report any security vulnerabilities or bugs that they discover while using our products.

## Submitting a Bug Report

If you discover a security vulnerability or bug, please follow the steps below to submit a bug report:

1. **Verify the Bug**
   Before submitting a bug report, please verify that the bug is reproducible and not a one-time occurrence.
2. **Document the Bug**
   Document the bug by providing a detailed description of the issue and steps to reproduce it. Please also include any relevant error messages, screenshots, or code snippets that may help us understand the issue.
3. **Submit the Bug Report**
   Submit the bug report by creating a new issue in our GitHub repository. Please use the Bug Report template provided in the repository and provide as much detail as possible. If you're not sure whether the issue you've found qualifies as a security vulnerability, please submit it anyway and we will review it.

## Submitting a Pull Request

If you discover a security vulnerability or bug and have the skills to provide a fix, please follow the steps below to submit a pull request:

1. **Fork the Repository**
   Fork the repository to your own GitHub account and create a new branch for your changes.
2. **Make the Changes**
   Make the necessary changes to the code to fix the bug.
3. **Test the Changes**
   Test your changes to ensure that they fix the bug and don't introduce any new issues.
4. **Submit the Pull Request**
   Submit the pull request by creating a new pull request in our GitHub repository. Please use the Pull Request template provided in the repository and provide as much detail as possible about the changes you've made.

## Bug Bounty Rewards

We may also reward users who responsibly disclose security vulnerabilities or bugs through our bug bounty program. The rewards vary depending on the severity of the issue and the quality of the bug report or pull request.

## Contact Information

If you have any questions or concerns about our bug bounty program, please contact us at hello@webb.tools. We take all bug reports seriously and will do our best to respond to your report as soon as possible.

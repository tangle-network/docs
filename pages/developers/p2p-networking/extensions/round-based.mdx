# Round-Based Networking Extension

SDK source (GitHub): https://github.com/tangle-network/blueprint/tree/main/crates/networking/extensions/round-based

This extension provides the `RoundBasedNetworkAdapter`, which bridges a `NetworkServiceHandle` with the `round-based`
protocol framework. See the [P2P usage guide](/developers/p2p-networking/usage) for a minimal integration example.
